# 모모

{% hint style="info" %}
방랑하는 베르더 소녀인 모모는 어둠의 빛의 비인도적인 실험으로 인해 몸의 일부에 끔찍한 상처가 있다. 그녀는 항상 어딘가에 홀린 듯한 표정이였고 반응이 느렸다. 실험의 후유증인지 아니면 다른 이유인지는 모르겠지만, 모모는 항상 사람들에게 이상하게도 초연함을 느끼게 했다.
